# Invoice Memory AI

Invoice Memory AI is a TypeScript-based intelligent invoice processing engine that uses a memory-driven decision system to analyze invoices.

## Features
- Invoice normalization
- Memory-based decision making
- Confidence score calculation
- Audit trail generation
- Modular and type-safe architecture

## Tech Stack
- TypeScript
- Node.js

## Project Structure
- src/engine – core decision and processing logic
- src/memory – memory store and confidence logic
- src/data – sample invoice and memory data
